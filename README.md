# Photos Album Generator

一个本地运行的相册生成管理系统，支持上传照片、选择相册风格、编辑照片位置和文本内容，并导出为 PDF 或图片。

## 功能

- 上传多张图片并自动生成相册
- 支持相册集页数设置
- 每一页可以单独选择风格
- 内置 10 种风格背景：复古、现代、可爱、冷酷、爱情、梦幻、古风、赛博、唯美、清新
- 使用 `assets/backgrounds` 中的高清图片作为相册背景
- 支持自定义照片位置、大小、旋转角度
- 支持删除照片、重置当前页布局
- 支持新增自定义文本框并编辑文本内容、位置、颜色、字号和旋转
- 支持导出当前页 PNG
- 支持逐页导出图片
- 支持整本相册导出为 PDF
- 新增自由创作模式，初始为白纸画布
- 自由创作模式支持 A3、A4、B5 和手动输入尺寸
- 自由创作模式支持上传背景图替换白纸
- 自由创作模式中照片可自由拖动、缩放，并支持 360 度旋转

## 使用方法

直接打开 `index.html` 即可使用。

1. 点击“上传图片”，选择要制作相册的照片。
2. 在“相册集”中设置页数。
3. 点击底部缩略图切换页面。
4. 在“当前页风格”中为每一页选择背景风格。
5. 开启“照片位置”开关后，可拖动照片并调整大小、旋转。
6. 点击“文本框 - 新增”，可以添加并编辑自定义文字。
7. 使用顶部按钮导出当前页、逐页图片或整本 PDF。

## 自由创作模式

1. 在左侧“模式”中切换到“自由创作”。
2. 在“自由画布”中选择 A3、A4、B5，或选择手动输入并填写宽高。
3. 不上传背景时，画布是一张白纸。
4. 点击“上传背景”后，背景图会替换白纸画布。
5. 在自由创作模式下上传照片，照片会作为独立素材放到画布上。
6. 选中照片后，可以拖动位置、调整大小，并使用旋转滑块进行 0 到 360 度旋转。
7. 自由创作画布同样支持 PNG 和 PDF 导出。
